# Smart Note App

A clean, minimalist, and responsive note-taking application built with vanilla JavaScript, HTML5, and CSS3.

## Live Preview

🔗 [View the live app here](https://ahdevworker03.github.io/Notes-App/)

## Features

- **Create & Edit**: Easily add new notes or modify existing ones via a sleek modal interface.
- **Persistent Storage**: Your notes are saved to your browser's local storage, so they persist even after a page refresh.
- **Character Limit**: Integrated 300-character limit per note with a real-time counter.
- **Responsive Design**: A dynamic grid layout that looks great on desktops, tablets, and mobile devices.
- **Smart Feedback**: Visual cues (like shake animations) for empty submissions to ensure a smooth user experience.
- **Newest First**: Your latest thoughts always appear at the top of the list.

## How to Use

Since this is a vanilla web application, there are no dependencies or build steps required.

1.  **Clone or Download** this repository to your local machine.
2.  Locate the `index.html` file in the root directory.
3.  **Open `index.html`** in any modern web browser (Chrome, Firefox, Edge, Safari, etc.).

## Technologies Used

- **HTML5**: For semantic structure.
- **CSS3**: For styling, including CSS Grid, Flexbox, and Custom Properties (Variables).
- **JavaScript (ES6+)**: For application logic and state management.
